# Security Policy

## Intended use

Use this utility only on networks and systems you own or are explicitly authorized to assess. Start with passive mode on sensitive OT segments.

## Report sensitivity

Generated reports may contain IP addresses, MAC addresses, reverse-DNS names, interface names, usernames and workstation names. Reports are ignored by the repository `.gitignore` and should be reviewed before sharing.

## Vulnerability reports

Do not publish credentials, production topology or sensitive evidence in a public issue. Use a private GitHub security advisory when available.
